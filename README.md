# 🦇 Batman AI Discord Bot

A powerful AI-powered Discord bot that acts as Batman with 5 different personalities!

## Features
- 🎭 5 Unique Modes (Normal, Freaky, Angry, Mysterious, Hilarious)
- 💾 Persistent Memory (Remembers conversations)
- 🧠 Powered by Bytez AI (Qwen3-0.6B)
- 💬 Responds to mentions and DMs
- 🔄 Auto-save memory

## Commands
- `/mode` - Change Batman's personality
- `/status` - Check your current mode
- `/reset` - Clear conversation history
- `/remember` - Make Batman remember something
- `/forget` - Make Batman forget
- `/help` - Show help

## Setup
1. Add your Discord token to Railway variables
2. Deploy
3. Invite bot to your server

## Tech Stack
- Discord.py
- Bytez API
- Flask (web server)
